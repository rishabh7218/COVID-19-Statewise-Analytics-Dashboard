# COVID-19 Statewise Analytics Dashboard

## Project Overview

This project analyzes COVID-19 statewise data using Excel and Power BI. The dashboard provides insights into confirmed, recovered, and deceased cases across different Indian states.

## Tools Used

* Microsoft Excel
* Power BI

## Features

* KPI Cards
* State-wise Filter (Slicer)
* Top 10 States by Confirmed Cases
* Top 10 States by Recoveries
* Top 10 States by Fatalities

## Dataset Source

Kaggle COVID-19 Statewise Dataset

## Skills Demonstrated

* Data Analysis
* Data Visualization
* Dashboard Development
* Excel
* Power BI
* Pivot Tables
